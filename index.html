What is this?

jQuery.selection is a jQuery plugin that operate selection text/caret.

You can get selection text/caret position information or get selected text, replace/insert selected text.

How to use?

jQuery.selection plugin uses the jQuery, only.

Include these two javascript files in your header as follows.
